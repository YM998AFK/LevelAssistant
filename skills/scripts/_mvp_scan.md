## 4. 跨包复现的 Broadcast 子序列（桥段候选）

合计 9 个非冗余候选子序列（len>=3, 出现在>=2包）：

- **[L8]×2包(2次)**  `打开日晷 → 开启测算 → 测算输入 → 周期循环 → 测算分析 → 凸显图标 → 说结果 → 长老验算`  _来源: 练习5×1, 练习7×1_
- **[L8]×2包(2次)**  `开启测算 → 测算输入 → 周期循环 → 测算分析 → 凸显图标 → 说结果 → 长老验算 → 正确效果`  _来源: 练习5×1, 练习7×1_
- **[L7]×2包(2次)**  `传递失败 → 初始化 → 准备 → 机器人左转 → 机器人发射 → 屏幕显示 → 计算`  _来源: 挑战2×1, 挑战3×1_
- **[L7]×2包(2次)**  `传递失败 → 传递失败 → 传递失败 → 传递失败 → 传递失败 → 传递失败 → 传递失败`  _来源: 练习5×1, 练习7×1_
- **[L4]×4包(4次)**  `传递失败 → 运行 → 运行 → 运行`  _来源: 挑战2×1, 挑战3×1, 练习5×1, 练习7×1_
- **[L4]×3包(3次)**  `初始化 → judge → cin判断 → cout判断`  _来源: 挑战2×1, 挑战3×1, 挑战9-教师端×1_

- **[L4]×2包(2次)**  `传递失败 → 展示关卡效果 → 传递成功 → 传递成功`  _来源: 练习5×1, 练习7×1_
