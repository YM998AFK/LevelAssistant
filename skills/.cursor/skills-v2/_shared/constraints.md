## 全局规则（三条，所有 Agent 随时生效）

| # | 规则 | 说明 |
|---|------|------|
| M4 | **有歧义/资源缺口/不确定时，主动向设计师提问，不擅自假设** | 不确定 → 问，不猜 |
| M7 | **修改或生成关卡前，必须获得人类明确确认（"确认/可以/OK"），未获确认 = 强制停止** | 改之前必须等人说 OK |
| N10 | **禁止擅自假设剧情 / 单位 / 资源** | 不编造，不脑补 |

---

其余执行规则（verify_gates、validate、block构建、角色间距等）均内嵌在各模式的 `agents/` 文件对应执行步骤中，不在此集中列出。
