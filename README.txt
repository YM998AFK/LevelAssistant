# 关卡助手 LevelAssistant

基于 Claude AI 的盘古 3D 关卡辅助工具，支持修改关卡、新建关卡、资源搜索等工作流。

---

## 功能模式

| 模式 | 说明 |
|------|------|
| **✏ 修改关卡** | 上传现有关卡包，描述修改需求，AI 自动分析、改造并生成新包体 |
| **🆕 新建关卡** | 描述关卡玩法需求，AI 按规范从零生成编程关卡包 |
| **📖 新建剧情** | 描述剧情内容，AI 生成剧情关卡包 |
| **🔍 资源搜索** | 输入资源描述，AI 并行搜索角色/物件/场景/音效等资源并输出清单 |
| **💬 自由对话** | 与 AI 自由交流，无特定工作流约束 |

---

## 使用方式

### 修改关卡

1. 左侧面板上传**关卡包**（`.ws` 文件）
2. 填写修改需求（如"把第一排星星改为 4 颗"）
3. 点击「开始修改」
4. AI 自动完成分析 → 改造 → 审查，通过后出现下载卡
5. 下载修改完成的 `.zip` 包体，解压替换原文件

### 新建关卡 / 新建剧情

1. 左侧面板可上传参考文件（可选）
2. 填写关卡描述需求
3. 点击「生成新关卡」/「生成新剧情」
4. 审查通过后下载生成的包体

### 资源搜索

1. 左侧面板填写资源描述（如"需要一个跑步的小女孩角色"）
2. 点击「开始搜索」
3. AI 并行检索多类资源，输出带路径的资源清单

### 自由对话

直接在右侧输入框输入内容，回车发送，与 AI 自由交流。

---

## 输出目录

所有包体产出统一输出到：

```
LevelAssistant\output\
  ├── modify\      # 修改关卡产出
  └── new\         # 新建关卡/剧情产出
```

---

## 界面说明

- **顶部**：模式切换标签、「清空历史」按钮
- **左侧**：随模式变化的输入面板（文件上传、需求描述、执行按钮）
- **右侧**：多会话聊天区，展示 AI 思考过程、工具调用状态、最终回复及下载卡

---

## 运行环境

- Windows 10 / 11
- 直接运行 `dist\LevelAssistant.exe`，无需安装 Python

### 开发模式运行

```bash
# 安装依赖
pip install -r requirements.txt

# 启动
python main.py
```

### 重新打包

```bash
.venv\Scripts\pyinstaller.exe LevelAssistant.spec --noconfirm
```
